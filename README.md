# BASE-0.1
---------------------------------------------------------
Automatic Base Control System

## Controllers Used
- STM32F407VGT60
- STM32F103C8
- ESP32 WROOM
- Sony PS4 Joystick

## Drivers Used
- Four Omniwheels
- Four DC Gear Motor
- Two motor drivers driving two motors each

## Sensors Used
- Four hall rotary shaft encoders
- Three freewheels
- One infrared Sensor

## Algorithms Used
- Omniwheel Kinematics
- PID control

## Communication Used
- UART
- Bluetooth

## Materials Used
- Aluminium base frame
- Copper PCB Boards
- Ply to support PCB
